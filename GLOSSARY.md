# Evnetic OS Glossary

## Agent

Software entity capable of reasoning, planning and using tools.

---

## Application

User-facing software built on Evnetic OS.

---

## Capability

A functional ability exposed by a device or software module.

Examples:

* navigation.
* vision.
* audio.

---

## Device

Any physical machine managed by Evnetic OS.

Examples:

* rover.
* drone.
* robotic arm.

---

## Edge Runtime

Software executing locally on autonomous machines.

Responsible for:

* execution.
* communication.
* safety.

---

## Extension

External software module that adds functionality.

---

## Kairo

The intelligence layer of Evnetic OS.

Responsible for:

* reasoning.
* planning.
* interaction.

---

## Mission

A goal-oriented task assigned to a device.

Example:

"Inspect warehouse."

---

## Skill

An executable capability or action.

Example:

"Search object."

---

## Tool

A controlled interface used by Kairo to interact with Evnetic OS.

---

## Fleet

A collection of devices managed together.

---

## Control Plane

Cloud services responsible for management and coordination.

---

## Runtime

Software responsible for executing operations.

---

## Capability Registry

System responsible for discovering available capabilities.
